Create a batch job from a JSONL file you already uploaded with [`POST /v1/files`](/api-reference/batch/files-api).

1. Upload input JSONL with `purpose=batch` (see [Quickstart](/api-reference/batch/getting-started)).
2. Paste the returned `file-…` id into `input_file_id` below.
3. Set `endpoint` to `/v1/chat/completions` (must match every line in your JSONL).

Full reference: [Batches API](/api-reference/batch/batches-api).

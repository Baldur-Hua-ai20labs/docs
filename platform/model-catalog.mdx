## Detailed model cards

### `zlm-v1-iab-classify-edge`

**Task:** Text Classification  
**Provider:** ZeroGPU  
**Best for:** Real-time IAB taxonomy mapping in ad-tech/content pipelines.

ZeroGPU's IAB taxonomy classifier is purpose-built for real-time content categorization at the edge. It maps text to the IAB Content Taxonomy, the industry standard used across programmatic advertising, enabling instant low-latency content classification without roundtrips to a centralized server. At 90M parameters on ONNX, it is optimized for high-volume, sub-millisecond classification that ad tech and content platforms demand.

**Specs:** `Parameters: 90M` · `Context: 400` · `Architecture: ONNX`  
**Pricing:** `$0.25 / 1M input` · `$0.40 / 1M output`

### `zlm-v1-iab-classify-edge-enriched`

**Task:** Text Classification  
**Provider:** ZeroGPU  
**Best for:** Pipelines that need category + additional content signals.

The enriched variant of ZeroGPU's IAB classifier goes beyond standard taxonomy labels by producing multi-signal outputs, giving you richer and more actionable classification data per inference call. It uses the same 90M-parameter ONNX architecture as the base classifier, so it runs at the same edge-native speed profile.

**Specs:** `Parameters: 90M` · `Context: 400` · `Architecture: ONNX`

## Using a model from the catalog in an API request

Once you’ve picked a model card, copy the **Model ID** and set it in your request:

```json
{
  "model": "zlm-v1-iab-classify-edge"
}
```

The rest of the request (headers and `input`) follows the standard `POST /v1/responses` format.
